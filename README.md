# Blocks Community Catalog

Community-contributed block catalogs for the [Blocks Editor](https://github.com/linucs/vscode-blockly) VS Code extension.

## Using catalogs

The Blocks Editor extension includes a **Community Catalog** browser in the Activity Bar. Browse, search, and download catalogs directly into your project.

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) for how to add your own block catalogs.

## Structure

```
catalogs/
  <vendor>/
    <component>.yaml
```

Each YAML file defines blocks for a specific hardware component or library, following the [block-catalog v1 schema](https://github.com/linucs/vscode-blockly/blob/main/src/catalog/block-catalog_v1.schema.json).

## Index

`index.json` is auto-generated by a GitHub Action on every push to `main`. The Blocks Editor extension fetches this index to populate the catalog browser.
